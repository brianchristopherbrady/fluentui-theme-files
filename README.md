# Fluent Theme Files

Generates exports for Fluent UI theme tokens and alias files in CSS, SCSS, and TS.  

__npm package:__ 
https://www.npmjs.com/package/@brianbrady/fluentui-theme-files


# Installation
Using npm:

```powershell
    npm i @brianbrady/fluentui-theme-files
```

Run:
```powershell
    ts-node-esm index.ts
```
